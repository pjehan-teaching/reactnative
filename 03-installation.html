Expo

Expo est un SDK (Software Development Toolkit) permettant de développer rapidement des applications iOS ou Android.

Le projet peut démarrer directement sur un smartphone ou bien dans un émulateur Android / iOS.

Expo permet d'accéder aux composants hardware du smartphone (caméra, capteurs, gyroscope...).

Expo

npm install -g expo-cli
expo init my-project
  • Managed Workflow : Simple d'utilisation et ne nécessite pas d'installer Xcode ou Android Studio. Ne permet d'écrire que du code JS.
  • Bare Workflow : Plus compliqué à configurer mais permet un contrôle total de l'application. Permet d'utiliser l'API de Expo mais avec une installation et une configuration manuelle.
