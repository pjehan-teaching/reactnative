TextInput

Le composant <TextInput> permet de gérer les saisies des utilisateurs :

<TextInput onChangeText={text => this.setState({title: text})} value={this.state.title}/>
TextInput

Picker

Le composant <Picker> permet d'afficher une liste déroulante de choix.

Les choix sont affichés à l'aide du composant <Picker.Item> :

<Picker selectedValue={this.state.color} onValueChange={value => this.setState({color: value}) }>
  <Picker.Item label="Rouge" value="red" />
  <Picker.Item label="Vert" value="green" />
</Picker>

Switch

Le composant <Switch> permet d'afficher un bouton On / Off :

<Switch value={this.state.wifi} onValueChange={value => this.setState({wifi: value}) }>
