Les composants View et Text

Le composant <View> est le conteneur le plus basique. Il permet d'afficher ses éléments enfants et de définir leur mise en page.

Le composant <ScrollView> est un conteneur permettant à l'utilisateur de scroller.

Le composant <Text> permet quant à lui d'afficher du texte et de définir sa mise en forme.

<View>
  <Text>Hellow world!</Text>
</View>

Gestion des styles

Il n'existe pas de feuille de style pour gérer la mise en forme des composants.

Les styles sont définies dans des objets JavaScript via l'attribut style et utilisent une syntaxe camelCase pour les propriétés (ex: fontSize)

Il est possible de définir l'ensemble des styles d'un composant en utilisant la fonction StyleSheet.create :

const styles = StyleSheet.create({
  red: {
    color: 'red'
  },
  blue: {
    color: 'blue'
  }
});

<Text style={styles.red}>Texte rouge</Text>

Android StatusBar

Par défaut sur Android, l'application est affichée derrière la barre de status.

Il est possible de corriger cela en utilisant le composant StatusBar de ReactNative :

import { StatusBar } from 'react-native';

const styles = StyleSheet.create({
  container: {
    marginTop: StatusBar.currentHeight
  },
});

Positionnement

Il est possible de définir la taille d'un composant en définissant les attributs height et width.

Il est également possible d'utiliser la propriété flex dans les styles pour définir une taille en proportion.

Flex ne fonctionne que si l'élément parent à une taille définie

Liste de composants

Il est possible d'afficher une liste de composants en utilisant le composant <FlatList>. Ce composant permet d'afficher une liste de composants de manière uniforme en utilisant le lazy loading.

Il est également possible d'utiliser le composant <SectionList> pour afficher une liste d'éléments organisés par sections (lazy loading).

Contraiement aux composants FlatList ou SectionList, <ScrollView> affiche directement l'ensembles des composants de la liste ce qui peut impacter les performances de l'application.

Orientation

Le module Dimension permet de connaître les dimensions de l'écran.

Il est ainsi possible de savoir si l'écran est au format paysage (landscape) ou portrait :

import { Dimensions } from 'react-native';

const {height, width} = Dimensions.get('window');
const orientation = height > width ? 'portrait' : 'landscape';
